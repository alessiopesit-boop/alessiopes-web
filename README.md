# alessiopes.it

Sito commerciale personale di Alessio Pes — sviluppatore web e software freelance.

Realizzato in HTML/CSS/JS vanilla, senza framework né CMS. Deploy su Netlify/Cloudflare Pages con dominio `alessiopes.it`.

## Pagine

- `index.html` — Home
- `servizi.html` — Servizi e prezzi
- `preventivo.html` — Configuratore preventivo interattivo
- `progetti.html` — Portfolio progetti
- `chi-sono.html` — Bio
- `contatti.html` — Contatti (WhatsApp + email)
- `faq.html` — Domande frequenti
- `google.html` — Servizi Google (Ads, LSA, Profilo)
- `privacy.html` — Privacy & cookie
- `404.html` — Pagina non trovata

## Deploy

GitHub Pages: https://alessiopesit-boop.github.io/alessiopes-web

Push su `main` → deploy automatico. Release → versione taggata (gestita da release-please).

## Setup locale

```bash
npm run serve   # avvia server locale
```
